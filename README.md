# Safe Long-Prompt Generator

A **fully anonymized Python tool** for generating **long, safe prompts (≥24k tokens)** in multiple languages, intended as a **research artifact**.

---

## Features

* Supports **Polish, Turkish, Chinese, Hindi**
* Content categories: `household_tool`, `beverage_recipe`, `craft_technique`
* **Deterministic and reproducible** output
* Designed for **academic use**

---

## Installation

```bash
git clone https://github.com/yourusername/safe-long-prompt-generator.git
cd safe-long-prompt-generator
```

> Requires **Python 3.8+**; no extra dependencies.

---

## Usage

```bash
python3 prompt_generator.py --language polish --category household_tool
```

**Arguments**:

* `--language`: `polish`, `turkish`, `chinese`, `hindi`
* `--category`: `household_tool`, `beverage_recipe`, `craft_technique`
